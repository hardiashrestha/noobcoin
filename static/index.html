📚 How Blockchain Works

1. BLOCK → A container holding a list of transactions.
2. HASH → A unique SHA-256 fingerprint of a block's data. Change even 1 character → hash changes completely.
3. CHAIN → Every block stores the previous block's hash. Change block #2 → block #3's prevHash breaks → whole chain invalid.
4. MINING → Finding a hash that starts with N zeros by brute-forcing a nonce. Makes faking blocks computationally expensive.
5. WALLET → Your unique address on the network. Identified by name here (in real crypto, it's a public key).
6. NBC → NoobCoin. Your virtual currency on this chain. Free starter balance: 100 NBC 🎉
